# 🚀 نظام إدارة المشاريع والمهام المتقدم (Full Stack)
### مطور بواسطة: Eng. Eyad Mufleh ❤️

هذا المشروع هو "نسخة احترافية" مصممة لاختبار مهارات الـ Full Stack، يجمع بين قوة **Laravel 11** وكفاءة **React 18** مع لمسات تجربة مستخدم (UX) استثنائية.

---

## ✨ المميزات الاحترافية (تم تنفيذها بالكامل)
- **[x] لوحة إحصائيات ذكية (Dashboard):** عرض فوري لإجمالي المشاريع، المهام المكتملة، والمهام العالقة.
- **[x] شريط التقدم (Visual Progress):** حساب نسبة الإنجاز لكل مشروع حياً ومباشرة.
- **[x] محرك بحث لحظي:** بحث سريع في المشاريع بدون تأخير (Real-time Filtering).
- **[x] تجربة مستخدم فائقة (Premium UX):**
    - استخدام **Skeleton Loading** بدلاً من رسائل التحميل التقليدية.
    - رسائل تنبيه ذكية عند جلب البيانات توضح (عدد الناجح | عدد الفاشل).
- **[x] تصميم ثابت (Fixed Layout):** واجهة مستخدم تشبه التطبيقات السحابية مع ترويسة وقاع ثابتين.

---

## 📁 الهيكل التقني للمشروع
- **Backend:** Laravel 11.
- **Frontend:** React 18 + Ant Design v5 + Vite.
- **Database:** MySQL مع فهارس محسنة (Optimized Indexing).

---

## 🛠️ أفضل الممارسات البرمجية (Best Practices المنفذة)

### ⚡ الأداء والقابلية للتوسع (Performance & Scalability)
- **حل مشكلة N+1 Query:** تم استخدام `withCount()` لجلب إحصائيات المهام باستعلام واحد فقط، مما يمنع إرهاق السيرفر عند وجود آلاف المشاريع.
- **فهرسة قواعد البيانات (Indexing):** إضافة فهارس لحقول `project_id` و `status` لضمان سرعة البحث في أجزاء من الثانية.
- **تحجيم البيانات (Selective Select):** طلب الحقول الضرورية فقط من قاعدة البيانات لتوفير استخدام الذاكرة (RAM) وتقليل حجم البيانات عبر الشبكة.
- **الإدخال الجماعي (Bulk Insert):** برمجة الـ Seeders لإنشاء 3000 مشروع وعشرات الآلاف من المهام في ثوانٍ معدودة.

### 🏗️ جودة الكود (Code Quality)
- **مركزية الـ API (Service Pattern):** استخدام مكتبة **Axios** في ملف موحد لجميع الطلبات لضمان مبدأ DRY (Don't Repeat Yourself).
- **تكامل البيانات (Referential Integrity):** استخدام `onDelete('cascade')` لضمان نظافة قاعدة البيانات ومنع وجود بيانات يتيمة.
- **التحقق المزدوج (Double Validation):** تأمين البيانات عبر التحقق في الواجهة الأمامية (User Experience) وفي الواجهة الخلفية (Security).

## 🚀 تعليمات التشغيل السريع
1. `composer install` & `php artisan key:generate`
2. `php artisan migrate:fresh --seed` (لتوليد 3000 مشروع للاختبار).
3. `npm install` & `npm run dev`.
